contains()

Description:
The contains() method returns a Boolean value indicating whether a node is a descendant of a specified node.
A descendant can be a child, grandchild, great-grandchild, and so on.
A Boolean, indicating whether a node is a descendant of a specified node:
  • true - The node is a descendant.
  • false - The node is NOT a descendant.

Parameter Values

ParameterTypeDescription
node Node object Required. Specifies the node that may be contained by (a descendant of) a specified node.
