activeElement

Description:
The activeElement property returns the currently focused element in the document.
This property is read-only.
Notes:
For iOS, elements that are not <input type="text">, will not get focus assigned to them. The property will then return BODY or null.

Examples

<body onclick="exampleFunction()">

<div>
<p id="example"> </p>
</div>

</body>
function exampleFunction() {

let x = document.activeElement.tagName;
document.getElementById("example").innerHTML = x;

}
