onreset

Description:
The onreset event occurs when a form is reset.
Supported HTML tags: <form>.

Examples

Name:

<body>
<div>
<form onreset="eventOnresetFunc()">

Name: <input type="text">
<input type="reset">

</form>
<p class="replace"> </p>
<div>
</body>
function eventOnresetFunc() {

document.getElementsByClassName("replace")[0].innerHTML = "The form was reset!";

}
