blur()

Description:
The blur() method removes focus from the current window.
Notes:
This method makes a request to bring the current window to the background. It may not work as you expect in all browsers, due to different user settings.

Examples

function setWindowBlur() {

let myWindow = window.open("", "", "width=200, height=100");
myWindow.document.write("<p>A new window!</p>");
myWindow.blur();

}
