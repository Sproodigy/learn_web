fullscreenElement

Description:
The fullscreenElement property returns the current element that is displayed in fullscreen mode, or null when not in fullscreen.
Notes:
This property requires specific prefixes to work in different browsers (see Browser Support below).
Use the element.requestFullscreen() method to view an element in fullscreen mode.
Use the element.exitFullscreen() method to cancel fullscreen mode.
Prefix for different brousers: Chrome — 45.0 (webkit), IE — 11.0 (ms), Firefox — 47.0 (moz), Safari — 5.1 (webkit), Opera — 15.0 (webkit).
