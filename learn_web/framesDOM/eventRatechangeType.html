onratechange

Description:
The onratechange event occurs when the playing speed of the audio/video is changed (like when a user switches to a slow motion or fast forward mode).
This event is invoked by the playbackRate property of the Audio/Video Object, which sets or returns the current playback speed of an audio/video.
Supported HTML tags: <audio> and <video>

Examples

<body>
<button onclick="setPlaySpeed()"> Set the play speed slowly <button>

<video controls onratechange="eventOnratechangeFunc()">

<source src="../video/scolopendra_giant.mp4" type="video/mp4">
Your browser does not support the video element.

</video>

<p class="replace"> </p>
</body>
function setPlaySpeed() {

var x = document.getElementsByTagName("video")[0];
x.playbackRate = 0.3;

}

function eventOnratechangeFunc() {

document.getElementsByClassName("replace")[0].innerHTML = "The playing speed of the video was changed!";

}
