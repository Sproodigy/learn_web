length

Description:
The length property returns the number of nodes in a NamedNodeMap object.
A Node object's attributes is an example of a NamedNodeMap object.
This property is read-only.

Examples

Hint: max buttons = 5.

<div id="example> </div>
function returnLength() {

let butNum = prompt('Input button number:');

if (isNan(butNum)) {
alert('Please enter numbers!');
} else {

let len = document.getElementsByTagName('button')[butNum]attributes.length;
document.getElementById('example').innerHTML = len;

}
}
