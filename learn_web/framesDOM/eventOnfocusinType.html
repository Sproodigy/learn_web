onfocusin

Description:
The onfocusin event occurs when an element is about to get focus.
The onfocusin event is similar to the onfocus event. The main difference is that the onfocus event does not bubble. Therefore, if you want to find out whether an element or its child gets the focus, you should use the onfocusin event.
Supported HTML tags: ALL HTML elements, EXCEPT: <base>, <bdo>, <br>, <head>, <html>, <iframe>, <meta>, <param>, <script>, <style>, and <title>
Notes:
Although Firefox (52-) does not support the onfocusin event, you can find out whether a child of an element gets the focus or not, by using a capturing listener for the onfocus event (using the optional useCapture parameter of the addEventListener() method).
The onfocusin event may not work as expected in Chrome, Safari and Opera 15+ using the JavaScript HTML DOM syntax. However, it should work as an HTML attribute and by using the addEventListener() method.
