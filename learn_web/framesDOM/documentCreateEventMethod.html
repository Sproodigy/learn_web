createEvent()

Description:
The createEvent() method creates an event object.
The event can be of any legal event type, and must be initialized before use.

Parameter Values

ParameterDescription
type Required. A String that specifies the type of the event.

Possible values:

AnimationEvent
ClipboardEvent
DragEvent
FocusEvent
HashChangeEvent
InputEvent
KeyboardEvent
MouseEvent
PageTransitionEvent
PopStateEvent
ProgressEvent
StorageEvent
TouchEvent
TransitionEvent
UiEvent
WheelEvent
