removeNamedItem()

Description:
The removeNamedItem() method removes the node with the specified name in a NamedNodeMap object.

Parameter Values

ParameterTypeDescription
nodename String Required. The name of the node in the namedNodeMap you want to remove.

Examples

Hint: max buttons = 5.

Hint: class, type, style, onclick.

<div id="example> </div>
function fetchAttrName() {

let butNum = prompt('Input button number:');
let itemName = prompt('Input item name:');

if (isNan(butNum)) {
alert('Please enter numbers!');
} else {

let btn = document.getElementsByTagName('button')[butNum]attributes.length;
btn.attributes.removeNamedItem(itemName);
document.getElementById('example').innerHTML = itemName +' removed';

}
}
