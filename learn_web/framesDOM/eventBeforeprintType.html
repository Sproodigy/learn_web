onbeforeprint

Description:
The onbeforeprint event occurs when a page is about to be printed (before the print dialogue box appears).
The onbeforeprint event is the opposite of the onafterprint event.

Examples

<body onbeforeprint="beforeprintMessage()">

<p class="replace"> </p>

</body>
function beforeprintMessage() {

var x = document.getElementsByClassName("replace")[0];
x.innerHTML = "This document is now being printed.";

}
