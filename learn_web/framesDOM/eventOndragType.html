ondrag

Description:
The ondrag event occurs when an element or text selection is being dragged.
Drag and drop is a very common feature in HTML5. It is when you "grab" an object and drag it to a different location.
Notes:
To make an element draggable, use the global HTML5 draggable attribute.
Links and images are draggable by default, and do not need the draggable attribute.
While dragging an element, the ondrag event fires every 350 milliseconds.

Examples

<body>
<div>

<input type="text" oncopy="clipBoardEventOncopyFunc()" oncut="clipBoardEventOncutFunc()">
<input type="text" onpaste="clipBoardEventOnpasteFunc()">
<p class="replace"></p>

</div>
</body>

<script>
function clipBoardEventOncopyFunc() {

document.getElementsByClassName("replace")[0].innerHTML = "Text was copied!";

}

function clipBoardEventOncutFunc() {

document.getElementsByClassName("replace")[0].innerHTML = "The text has been cut!";

}

function clipBoardEventOnpasteFunc() {

document.getElementsByClassName("replace")[0].innerHTML = "Text has been inserted!";

}
</script>
