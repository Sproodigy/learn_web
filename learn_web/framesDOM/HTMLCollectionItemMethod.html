item()

Description:
The item() method returns the element at the specified index in an HTMLCollection.
The Elements are sorted as they appear in the source code, and the index starts at 0.

Parameter Values

ParameterTypeDescription
index Number Required. The index of the element you want to return.

Examples

Hint: max buttons = 6.

<div>

<p id="example"> </p>

</div>
functionshowHTMLCollectionItem() {

let butNum = prompt("Input button number:");

let but = document.getElementsByTagName("button").item(butNum).innerHTML;
document.getElementById('example').innerHTML = but;

}
functionshowHTMLCollectionItem() {

let butNum = prompt("Input button number:");

let but = document.getElementsByTagName("button")[butNum].innerHTML;
document.getElementById('example').innerHTML = but;

}
