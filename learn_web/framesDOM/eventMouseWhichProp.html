which

Description:
The which property returns a number that indicates which mouse button was pressed when a mouse event was triggered.
This property is mostly used together with the onmousedown event.
Possible values:
  1. No button.
  2. Left mouse button.
  3. Wheel button or middle button (if present).
  4. Right mouse button.
For a left-hand configured mouse, the return values are reversed.
This property is read-only.
Notes:
The values returned by this property are the same as those returned by the button property plus one.

Examples

<body onmousedown="eventMouseWhichFunc(event)">

<p class="replace"> </p>

</body>
function eventMouseWhichFunc(event) {

var x = document.getElementsByClassName("replace")[0];
x.innerText ="You pressed button: " + event.which;

}
