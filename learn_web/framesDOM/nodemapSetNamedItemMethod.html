setNamedItem()

Description:
The setNamedItem() method adds the specified node to the NamedNodeMap.
If the node already exists, it will be replaced, and the replaced node will be the return value, otherwise the return value will be null.
Notes:
Instead of working with attribute nodes, you could use the element.setAttribute() method to add an attribute with a value to an element.

Parameter Values

ParameterTypeDescription
node Node object Required. The node you want to add/replace in the NamedNodeMap collection.

Examples

class="setColor"

<div>

<p> class="setColor" </p>

</div>
.setColor {

color: "green";

}
function setNamedItem() {

let data = document.getElementsByTagName("p")[1];
let newAttr = document.createAttribute("class");
newAttr.value"setColor";
data.attributes.setNamedItem(newAttr);

}
