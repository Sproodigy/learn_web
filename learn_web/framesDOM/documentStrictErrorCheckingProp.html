strictErrorChecking

Description:
The strictErrorChecking property sets or returns whether a document has a strict error checking or not.
Default value is true.
Notes:
The elements in the collection are sorted as they appear in the source code.
The StrictErrorChecking property is not supported by most modern browsers.
