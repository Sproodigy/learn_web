onwaiting

Description:
The onwaiting event occurs when the video stops because it needs to buffer the next frame.
This event can also be used on <audio> elements, but it is mostly used for videos.
Supported HTML tags: <audio> and <video>.

Examples

<body>
<div>
<video controls onwaiting="eventOnwaitingFunc()">

<source src="../video/scolopendra_giant.mp4" type="video/mp4">
Your browser does not support the video element.

</video>

<div>
</body>
function eventOnwaitingFunc() {

document.getElementsByClassName("replace")[0].innerHTML = "Wait! I need to buffer the next frame!";
