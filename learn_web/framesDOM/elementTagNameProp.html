tagName

Description:
The tagName property returns the tag name of the element.
In HTML, the returned value of the tagName property is always in UPPERCASE.
This property is read-only.
Notes:
You can also use the nodeName property to return the tag name of an element. The difference is that nodeName also returns attribute nodes, text nodes, comment nodes.

Examples

<p> id="example1"> </p>
