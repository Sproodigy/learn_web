createAttribute()

Description:
The createAttribute() method creates an attribute with the specified name, and returns the attribute as an Attr object.
Notes:
Use the attribute.value property to set the value of the attribute.
Use the element.setAttributeNode() method to add the newly created attribute to an element.
Often, you will want to use the element.setAttribute() method instead of the createAttribute() method.

Parameter Values

ParameterTypeDescription
attributename Attr object Required. The name of the attribute you want to create.

Examples

let elem = document.getElementById("example");
let att = document.createAttribute("href");
att.value = "https://www.w3schools.com";
