getAttributeNode()

Description:
The getAttributeNode() method returns the attribute node with the specified name of an element, as an Attr object.
Notes:
Use the attribute.value property to return the value of the attribute node.
Use the getAttribute() method if you just want to return the attribute value.

Parameter Values

ParameterTypeDescription
attributename String Required. The name of the attribute you want to return.
