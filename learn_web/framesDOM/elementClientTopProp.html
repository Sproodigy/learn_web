clientTop

Description:
The clientTop property returns the width of the top border of an element, in pixels.
This property does not include the element's top padding or the top margin.
This property is read-only.
Notes:
You can also use the style.borderTopWidth property to return the width of an element's left border.
To return the width of the left border of an element, use the clientLeft property.
