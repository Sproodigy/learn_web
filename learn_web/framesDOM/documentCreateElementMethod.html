createElement()

Description:
The createElement() method creates an Element Node with the specified name.
Notes:
Use the createTextNode() method to create a text node.
After the element is created, use the element.appendChild() or element.insertBefore() method to insert it to the document.

Parameter Values

ParameterTypeDescription
nodename String Required. The name of the element you want to create.

Examples

Some paragraph.

<div id="example">

<p> Some paragraph. </p>

</div>
function createNewElement() {

let para = document.createElement("p");
let t = document.createTextNode("This is a new paragraph.");
para.appendChild(t);
document.getElementById("example").appendChild(para);

}
