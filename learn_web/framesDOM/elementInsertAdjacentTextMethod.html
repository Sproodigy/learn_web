insertAdjacentText()

Description:
The insertAdjacentText() method inserts a the specified element into a specified position.

Parameter Values

ParameterTypeDescription
position String Required. A position relative to the element.

Legal values:
"afterbegin" - After the beginning of the element (as the first child).
"afterend" - After the element.
"beforebegin" - Before the element.
"beforeend" - Before the end of the element (as the last child).
element HTML Element The element you want to insert.

Examples

Banana Kiwi Apple
<div>

<span class="fruit"> Banana </span>
<span class="fruit"> Kiwi </span>
<span class="fruit"> Aple </span>

</div>

var elem = document.getElementsByClassName("fruit")[1];
