hasChildNodes()

Description:
The hasChildNodes() method returns true if the specified node has any child nodes, otherwise false.
Whitespace inside a node is considered as text nodes, so if you leave any white space or line feeds inside an element, that element still has child nodes.

Examples

  • Banana
  • Apple

<div>

<ul id="example">
<li>Banana</li>
<li>Apple</li>
</ul>
<p class="replace"> </p>

</div>
function checkChildNodes() {

var node = document.getElementById("example").hasChildNodes();
document.getElementsByClassName("replace")[0].innerHTML = node;

}
