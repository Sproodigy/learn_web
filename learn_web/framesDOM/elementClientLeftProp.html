clientLeft

Description:
The clientLeft property returns the width of the left border of an element, in pixels.
This property does not include the element's left padding or the left margin.
This property is read-only.
Notes:
You can also use the style.borderLeftWidth property to return the width of an element's left border.
For Chrome, Firefox and Opera on Windows, if the element has a scrollbar (set by the CSS overflow property), and if the text direction of that element is right-to-left (set by the CSS direction property), this property includes the width of the vertical scrollbar, together with the left border width. In IE, however, it returns "0". On Mac OS, using Safari, Chrome, Firefox or Opera, it returns only the left border width.
