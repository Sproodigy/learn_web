onloadstart

Description:
The onloadstart event occurs when the browser starts looking for the specified audio/video. This is when the loading process starts.
Supported HTML tags: <audio> and <video>

Examples

<body>
<video controls onloadstart="eventOnloadstartFunc()">

<source src="../video/scolopendra_giant.mp4" type="video/mp4">
Your browser does not support the video element.

</video>

</body>
