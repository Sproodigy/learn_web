namedItem()

Description:
The namedItem() method returns the element with the specified ID, or name, in an HTMLCollection.

Parameter Values

ParameterDescription
name Required. The value of the id attribute, or name attribute, of the element you want to return.

Examples

Example text.

Hint: item name = test.

<div>

<p id="example"> </p>

</div>
functionshowHTMLCollectionNamedItem() {

let itemName = prompt("Input p number:");

let p = document.getElementsByTagName("p").namedItem(itemName).innerHTML;
document.getElementById("example").innerHTML = p;

}
functionshowHTMLCollectionNamedItem() {

let itemName = prompt("Input p number:");

let p = document.getElementsByTagName("p")[itemName].innerHTML;
document.getElementById("example").innerHTML = p;

}
