screenTop

Description:
The screenLeft and screenTop properties returns the x (horizontal) and y (vertical) coordinates of the window relative to the screen.
This property is read-only.

Examples

<div>

<p id="example"> </p>

</div>
function showWindowScreenTop() {

let myWindow = window.open("", "myWindow", "width="200", height=300");
document.getElementById("example").outerHTML = "The position of the window to the top: " + myWindow.screenTop;

}
