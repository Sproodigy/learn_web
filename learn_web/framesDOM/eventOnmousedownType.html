onmousedown

Description:
The onmousedown event occurs when a user presses a mouse button over an element.
Events related to the left and middle mouse button:
  • onmousedown
  • onmouseup
  • onclick
Events related to the right mouse button:
  • onmousedown
  • onmouseup
  • oncontextmenu
Supported HTML tags:
All HTML elements, EXCEPT: <base>, <bdo>, <br>, <head>, <html>, <iframe>, <meta>, <param>, <script>, <style>, and <title>.
