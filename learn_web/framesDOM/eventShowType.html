onshow

Description:
The onshow event occurs when the audio/video has been started or is no longer paused.
The onpause event occurs when the audio/video has been paused.
Supported HTML tags: <audio> and <video>

Examples

<body>
<video controls onpause="eventOnpauseFunc()" onshow="eventOnplayFunc()">

<source src="../video/scolopendra_giant.mp4" type="video/mp4">
Your browser does not support the video element.

</video>

<p class="replace"> </p>
</body>
function eventOnpauseFunc() {

document.getElementsByClassName("replace")[0].innerHTML = "The video was paused!";

}

function eventOnplayFunc() {

document.getElementsByClassName("replace")[0].innerHTML = "The video was played!";

}
