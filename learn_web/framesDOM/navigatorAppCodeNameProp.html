appCodeName

Description:
The appCodeName property returns the code name of the browser.
Notes:
All modern browsers returns "Mozilla", for compatibility reasons.
This property is read-only.

Examples

</div>

let x = "Browser CodeName: " + navigation.appCodeName + ".";
