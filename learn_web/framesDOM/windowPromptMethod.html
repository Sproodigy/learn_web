prompt()

Description:
The prompt() method displays a dialog box that prompts the visitor for input.
A prompt box is often used if you want the user to input a value before entering a page.
The prompt() method returns the input value if the user clicks "OK". If the user clicks "cancel" the method returns null.
Notes:
When a prompt box pops up, the user will have to click either "OK" or "Cancel" to proceed after entering an input value. Do not overuse this method, as it prevents the user from accessing other parts of the page until the box is closed.

Parameter Values

ParameterTypeDescription
text String Required. The text to display in the dialog box.
defaultText String Optional. The default input text.
