onmouseleave

Description:
The onmouseleave event occurs when the mouse pointer is moved out of an element.
This event is often used together with the onmouseenter event, which occurs when the mouse pointer is moved onto an element.
Supported HTML tags:
All HTML elements, EXCEPT: <base>, <bdo>, <br>, <head>, <html>, <iframe>, <meta>, <param>, <script>, <style>, and <title>.
Description:
The onmouseleave event is similar to the onmouseout event. The only difference is that the onmouseleave event does not bubble (does not propagate up the document hierarchy).
