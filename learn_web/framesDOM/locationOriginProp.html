origin()

Description:
The origin property returns the protocol, hostname and port number of a URL.
Notes:
If the port number is not specified in the URL (or if it is the scheme's default port - like 80, or 443), some browsers will not display the port number.
This property is read-only.

Examples

<div>

<p id="example"> </p>

</div>
function showLocationOrigin() {

let origin = location.origin;
document.getElementById("example").innerHTML = origin;

}
