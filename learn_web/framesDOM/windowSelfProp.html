self

Description:
The self property returns the current window.
The self property is often used in comparisons.
This property is read-only.

Examples

function compareWindows() {

if (window.top != window.self) {
document.getElementById("example").outerHTML = "This window is not the topmost window! Am I in a frame?";
} else {
document.getElementById("example").outerHTML = "This window is the topmost window!";
}

}
