close()

Description:
The close() method closes the output stream previously closeed with the document.open() method, and displays the collected data in this process.

Examples

function closeDocumentInWindow() {

let w = window.open("", "", "width=200, height=200");
w.document.open();
w.document.write("<h1>Here I'm!"</h1>)
w.document.close()

}
