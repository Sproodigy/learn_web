btoa()

Description:
The btoa() method encodes a string in base-64.
This method uses the "A-Z", "a-z", "0-9", "+", "/" and "=" characters to encode the string.

Parameter Values

ParameterDescription
str Required. The string to be encoded.

Examples

String: Hello world!
