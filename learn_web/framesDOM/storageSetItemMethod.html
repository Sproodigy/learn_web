setItem(keyname, value)

Description:
The setItem() method sets the value of the specified Storage Object item.
The setItem() method belongs to the Storage Object, which can be either a localStorage object or a sessionStorrage object.

Parameter values

ValueDescription
keyname Required. A String specifying the name of the key you want to set the value of.
value Required. A String specifying the value of the key you want to set the value of.

Examples

<div>

<p id="example"> </p>

</div>
function setItem() {

sessionStorage.setItem("fruit", "Banana");

}
