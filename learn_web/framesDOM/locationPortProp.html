port()

Description:
The port property sets or returns the port number the server uses for a URL.
Notes:
If the port number is not specified in the URL (or if it is the scheme's default port - like 80, or 443), some browsers will display 0 or nothing.

Property Values

ValueTypeDescription
port String Specifies the port number of the URL.

Examples

<p id="example"> </p>

</div>
