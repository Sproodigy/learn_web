namespaseURI

Description:
The namespaceURI property returns the URI of the specified node's namespce.
This property is read-only.
Notes:
An element node inherits the namespace of its parent node, therefore, all elements in an XHTML document has the namespaceURI "http://www.w3.org/1999/xhtml".

Examples

<div id="myDiv">

<p> id="example"> </p>

</div>
function getNamespaseURI() {

var x = document.documentElement.namespaceURI;
document.getElementsByClassName("example").innerHTML = x;

}
