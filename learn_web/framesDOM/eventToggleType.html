ontoggle

Description:
The ontoggle event occurs when the user opens or closes the <details> element.
The <details> element specifies additional details that the user can view or hide on demand.
Supported HTML tags: <details>.
Notes:
The details element and the ontoggle event is not supported in Internet Explorer or Safari 5 (and earlier).

Examples

Sweet fruits

Banana

Apple

Peach

<body>
<details ontoggle="eventOntoggleFunc()">

<summary>Sweet fruits</summary>
<p>Banana</p>
<p>Apple</p>
<p>Peach</p>

</details>

<p class="replace"> </p>
</body>
function eventOntoggleFunc() {

document.getElementsByClassName("replace")[0].innerHTML = "The ontoggle event occured!";

}
