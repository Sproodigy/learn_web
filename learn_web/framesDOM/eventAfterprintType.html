onafterprint

Description:
The onafterprint event occurs when a page has started printing, or if the print dialogue box has been closed.
The onafterprint event is the opposite of the onbeforeprint event.

Examples

<body onafterprint="afterprintMessage()">

<p class="replace"> </p>

</body>
function afterprintMessage() {

var x = document.getElementsByClassName("replace")[0];
x.innerHTML = "This document is now being printed.";

}
