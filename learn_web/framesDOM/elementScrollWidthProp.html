scrollWidth

Description:
The scrollWidth property returns the entire width of an element in pixels, including padding, but not the border, scrollbar or margin.
The scrollWidth and scrollHeight properties return the entire width and width of an element, including the width and width that is not viewable (because of overflow).
This property is read-only.
Notes:
To add scrollbars to an element, use the CSS overflow property.
