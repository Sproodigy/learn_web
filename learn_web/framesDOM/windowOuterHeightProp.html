outerHeight

Description:
The outerHeight property returns the outer height of the browser window, including all interface elements (like toolbars/scrollbars).
This property is read-only.

Examples

<div>

<p id="example"> </p>

</div>
function showOuterHeight() {

let h = window.outerHeight;
document.getElementById("example").outerHTML = "Outer height: " + h;

}
