error

Description:
The onerror event is triggered if an error occurs while loading an external file (e.g. a document or an image).
When used on audio/video media, related events that occurs when there is some kind of disturbance to the media loading process, are:
  • onabort
  • onstalled
  • onsuspend
Supported HTML tags: <img>, <input type="image">, <object>, <link>, and <script>.

Examples

<div>

<img src="/vasserman_big.png" onerror="onerrorMessage()">
<p class="replace"> </p>

</div>
function onerrorMessage() {

var x = document.getElementsByClassName("replace")[0];
x.innerHTML = "The image could not be loaded.";

}
