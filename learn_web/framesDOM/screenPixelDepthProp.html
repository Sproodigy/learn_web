pixelDepth

Description:
The pixelDepth property returns the color resolution (in bits per pixel) of the visitor's screen.
Notes:
The pixelDepth property is not supported in Internet Explorer 9 and earlier versions. However, the colorDepth property represents the same thing as the pixelDepth property. Since all major browsers support colorDepth, use that property instead.
