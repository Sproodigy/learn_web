onresize

Description:
The onresize event occurs when the browser window has been resized.
Supported HTML tags: <body>
Notes:
To get the size of an element, use the clientWidth, clientHeight, innerWidth, innerHeight, outerWidth, outerHeight, offsetWidth and/or offsetHeight properties.

Examples

Try to resize the browser window to display the windows height and width (Ctrl+ and Ctrl-).

<body onresize="eventOnresizeFunc()">
<p> Try to resize the browser window to display the windows height and width (Ctrl+ and Ctrl-).
</p> <p class="replace"> </p>
</body>
function eventOnresizeFunc() {

var w = window.outerWidth;
var h = window.outerHeight;
var txt = "Window size: width="+ w + ", height="+ h;
document.getElementsByClassName("replace")[0].innerHTML = txt;

}
