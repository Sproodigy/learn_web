onsearch

Description:
The onsearch event occurs when a user presses the "ENTER" key or clicks the "x" button in an <input> element with type="search".
Supported HTML tags: <input>.
Notes:
The onsearch event is not supported in Internet Explorer, Firefox or Opera 12 and earlier versions.

<div>

Name: <input type="search" id="example" onsearch="eventOnsearchFunc()">
