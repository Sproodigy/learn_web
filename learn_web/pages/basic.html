Basic

Doctype

Notes:
Always add the declaration to your HTML documents, so that the browser knows what type of document to expect.
The declaration must be the very first thing in your HTML document, before the <html> tag.
The declaration is not an HTML tag; it is an instruction to the web browser about what version of HTML the page is written in.
The declaration is NOT case sensitive.
To check if the HTML of your Web documents is valid, go to W3C's validation service.

Syntax

<!DOCTYPE html>
