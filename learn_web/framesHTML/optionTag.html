Forms and input

<option>

Description:
The <option> tag defines an option in a select list.
<option> elements go inside a <select> or <datalist> element.
Notes:
The <option> tag can be used without any attributes, but you usually need the value attribute, which indicates what is sent to the server.
If you have a long list of options, you can group related options with the <optgroup> tag.

Attributes

AttributeValueDescription
<disabled> disabled Specifies that an option should be disabled.
<label> text Specifies a shorter label for an option.
<selected> selected Specifies that an option should be pre-selected when the page loads.
<value> text Specifies the value to be sent to a server.

disabled

Descriptions:
The disabled attribute is a boolean attribute.
When present, it specifies that an option should be disabled.
A disabled option is unusable and un-clickable.
The disabled attribute can be set to keep a user from selecting the option until some other condition has been met (like selecting a checkbox, etc.). Then, a JavaScript is required to remove the disabled value, and make the option selectable.

Examples

<select>

<option value="banana" > Banana </option>
<option value="apple"> Apple </option>
<option value="orange"> Orange </option>
<option value="kiwi"> Kiwi </option>

</select>

label

Descriptions:
The label attribute specifies a shorter version of an option.
The shorter version will be displayed in the drop-down list.
Notes:
The label attribute is not supported in Firefox.

Attributes

ValueDescription
text A shorter version for the option.

Examples

<select>

<option label="Banana"> Banana (very sweet fruit) </option>
<option label="Apple"> Apple (sour and sweet fruit) </option>
<option label="Orange"> Orange (sweet fruit) </option>
<option label="Kiwi"> Kiwi (sweet fruit) </option>

</select>

selected

Descriptions:
The selected attribute is a boolean attribute.
When present, it specifies that an option should be pre-selected when the page loads.
The pre-selected option will be displayed first in the drop-down list.
Notes:
The selected attribute can also be set after the page loads, with a JavaScript.

Examples

<select>

<option value="banana"> Banana </option>
<option value="apple"> Apple </option>
<option value="orange" > Orange </option>
<option value="kiwi"> Kiwi </option>

</select>

value

Descriptions:
The value attribute specifies the value to be sent to a server when a form is submitted.
The content between the opening <option> and closing </option> tags is what the browsers will display in a drop-down list. However, the value of the value attribute is what will be sent to the server when a form is submitted.
Notes:
If the value attribute is not specified, the content will be passed as a value instead.

Attributes

ValueDescription
value The value to be sent to the server.

Examples

<form action="example.php">
<select name="fruits">

<option value="banana"> Banana </option>
<option value="apple"> Apple </option>
<option value="orange"> Orange </option>
<option value="kiwi"> Kiwi </option>

</select>
<input type="submit" value="Submit">
</form>
