<form> action

Description:
The action attribute specifies where to send the form-data when a form is submitted.
In HTML5, the action attribute is no longer required.
Attributes
ValueDescription
URL Where to send the form-data when the form is submitted.
An absolute URL - points to another web site (like src="http://www.example.com/example.php").
A relative URL - points to a file within a web site (like src="example.php").
