<src>

Description:
The src attribute specifies the location (URL) of the audio file.
To play the audio file in Internet Explorer and Safari, we must use an MP3 file.
To make it work in all browsers â€” use <source> elements inside the <audio> element. <source> elements can link to different audio files. The browser will use the first recognized format.
Note:
The .ogg fileformat is not supported in IE and Safari.
Attributes
ValueDescription
URL The URL of the audio file.
An absolute URL - points to another web site (like src="http://www.example.com/example.ogg").
A relative URL - points to a file within a web site (like src="example.ogg").

Examples

<audio src="?" controls>
Your browser does not support the audio element.
</audio>
