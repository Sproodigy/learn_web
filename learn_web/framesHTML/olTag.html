Lists

<ol>

Descriptions:
The <ol> tag defines an ordered list. An ordered list can be numerical or alphabetical.

Attributes

AttributeValueDescription
compact compact Not supported in HTML5.
Specifies that the list should render smaller than normal.
reversed reversed Specifies that the list order should be descending (9,8,7...).
start number Specifies the start value of an ordered list.
type disc
square
circle		 Not supported in HTML5.
Specifies the kind of marker to use in the list.

compact

Descriptions:
The compact attribute is a boolean attribute.
When present, it specifies that the list should render smaller than normal, by reducing the space between lines and the indentation of the list.
Notes:
The compact attribute of <ul> is not supported in HTML5. Use CSS instead.

Examples

  1. Coffee
  2. Tea
  3. Milk
<ol compact>

<li>Coffee</li>
<li>Tea</li>
<li>Milk</li>

</ol>

reversed

Descriptions:
The reversed attribute is a boolean attribute.
Notes:
The reversed attribute of the ol tag is not supported in IE and Safari 5.1 (and earlier versions).

Examples

  1. Coffee
  2. Tea
  3. Milk
  4. Cacao
<ol reversed>

<li>Coffee</li>
<li>Tea</li>
<li>Milk</li>

</ol>

start

Descriptions:
The start attribute specifies the start value of the first list item in an ordered list.

Attributes

ValueDescription
number Specifies the start value of the first list item in the ordered list.

Examples

  1. Coffee
  2. Tea
  3. Milk
  4. Cacao
<ol type="I" start="10">

<li>Coffee</li>
<li>Tea</li>
<li>Milk</li>

</ol>

type

Descriptions:
The type attribute specifies the kind of marker to use in the list (letters or numbers).

Attributes

ValueDescription
1 Default. Decimal numbers (1, 2, 3, 4).
a Alphabetically ordered list, lowercase (a, b, c, d).
A Alphabetically ordered list, uppercase (A, B, C, D).
i Roman numbers, lowercase (i, ii, iii, iv).
I Roman numbers, uppercase (I, II, III, IV).

Examples

  1. Coffee
  2. Tea
  3. Milk
  4. Cacao
<ol type="1">

<li>Coffee</li>
<li>Tea</li>
<li>Milk</li>

</ol>
