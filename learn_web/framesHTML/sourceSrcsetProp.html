<srcset>

Description:
The srcset attribute specifies the URL of the image to use in different situations.
This attribute is required when <source> is used in <picture>.
Attributes
ValueDescription
URL Specifies the URL of the image.
An absolute URL - points to another web site (like src="http://www.example.com/example.ogg").
A relative URL - points to a file within a web site (like src="example.ogg").

Examples

Resize browser window (Ctrl-, Ctrl+)

<picture>

<source media="(min-width: 1350px)" srcset="../images/vasserman_big.jpg">
<img src="../images/vasserman_thinking.gif" alt="vasserman">

</picture>
