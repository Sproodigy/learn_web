<canvas>

Descriptions:
The <canvas> tag is used to draw graphics, on the fly, via scripting (usually JavaScript).
The <canvas> tag is only a container for graphics, you must use a script to actually draw the graphics.
Note:
Any text inside the <canvas> element will be displayed in browsers that does not support <canvas>.
The canvas tag is not supported in Internet Explorer 8 and earlier versions.

Attributes

AttributeValueDescription
height pixels Specifies the height of the canvas.
width pixels Specifies the width of the canvas.

height

Descriptions:
The height attribute specifies the height of the <canvas> element, in pixels.
Notes:
Each time the height or width of a canvas is re-set, the canvas content will be cleared.

Attributes

ValueDescription
pixels Specifies the height of the canvas, in pixels (e.g. "100"). Default height is "150".

Examples

<div>

<canvas id="myCanvas" width="200" height="200" style="2px solid">

</div>

<script>

var c = document.getElementById("myCanvas");
var ctx = c.getContext("2d");
ctx.fillStyle = "#0890ff";
ctx.fillRect(50, 50, 100, 100);

</script>
