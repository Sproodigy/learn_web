Styles and Semantics

<section>

Descriptions:
The <section> tag defines sections in a document, such as chapters, headers, footers, or any other sections of the document.

Examples

Desined by: Uncle Sam.

The story about Uncle Sam.

Once upon a time, there was Uncle Sam...

Posted by: Uncle Sam.

Contact information: unclesam@gmail.com.

<header>

<h1>Desined by: Uncle Sam.</h1>

</header>
<section>

<h3>The story about Uncle Sam.</h3>
<p>Once upon a time, there was Uncle Sam...</p>

</section>
<footer>

<p>Posted by: Uncle Sam.</p>
<p>Contact information:
<a href="unclesam@gmail.com"> unclesam@gmail.com.</a>
</p>

</footer>

}

}
