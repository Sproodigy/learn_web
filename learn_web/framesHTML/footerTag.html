Styles and Semantics

<footer>

Descriptions:
The <footer> tag defines a<footer>for a document or section.
You can have several <footer> elements in one document.
A <footer> element typically contains:
  • authorship information
  • copyright information
  • contact information
  • sitemap
  • back to top links
  • related documents
Notes:
Contact information inside a <footer> element should go inside an <address> tag.

Examples

Desined by: Uncle Sam.

Posted by: Uncle Sam.

Contact information: unclesam@gmail.com.

<header>

<h1>Desined by: Uncle Sam.</h1>

</header>
<footer>

<p>Posted by: Uncle Sam.</p>
<p>Contact information:
<a href="unclesam@gmail.com"> unclesam@gmail.com.</a>
</p>

</footer>
