Lists

<li>

Descriptions:
The <li> tag defines a list item.
The <li> tag is used in ordered lists (<ol>), unordered lists (<ul>), and in menu lists (<menu>).

Attributes

Specifies which kind of bullet point will be used.
value number Specifies the value of a list item. The following list items will increment from that number (only for <ol> lists).

type

Descriptions:
The type attribute specifies the style of the bullet point of a list item in a list.
Notes:
The type attribute of <li> is not supported in HTML5. Use CSS instead.
All attributes can be applied to both lists.

Attributes

ValueDescription
For ordered list
1 Default. Decimal numbers (1, 2, 3, 4).
a Alphabetically ordered list, lowercase (a, b, c, d).
A Alphabetically ordered list, uppercase (A, B, C, D).
i Roman numbers, lowercase (i, ii, iii, iv).
I Roman numbers, uppercase (I, II, III, IV).
For unordered list
disc Default. A filled circle.
circle An unfilled circle.
square A filled square.

Examples

  1. Coffee
  2. Tea
  3. Milk
<ol compact>

<li>Coffee</li>
<li>Tea</li>
<li type="">Milk</li>

</ol>

value

Descriptions:
The value attribute sets the value of a list item. The following list items will increment from that number.
The value must be a number and can only be used in ordered lists (<ol>).

Examples

  1. Coffee
  2. Tea
  3. Milk
  4. Cacao
<ol reversed>

<li value="38">Coffee</li>
<li>Tea</li>
<li>Milk</li>

</ol>
