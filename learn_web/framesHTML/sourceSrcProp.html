<src>

Description:
The src attribute specifies the URL of the media file to play.
This attribute is required when <source> is used in <audio> and <video>.
Attributes
ValueDescription
URL Specifies the URL of the media file.
An absolute URL - points to another web site (like src="http://www.example.com/example.ogg").
A relative URL - points to a file within a web site (like src="example.ogg").

Examples

<audio controls>

<source src="..sounds/cat_mewing.mp3" type="audio/mpeg">
Your browser does not support the audio element.

</audio>
