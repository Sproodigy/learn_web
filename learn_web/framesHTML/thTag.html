Tables

<th>

Descriptions:
The tag defines a header cell in an HTML table.
The text in elements are bold and centered by default.
Notes:
Use the colspan and rowspan attribute to let the content span over multiple columns or rows!

abbr

Descriptions:
The abbr attribute specifies a shorter version of the content in a header cell.
The abbr attribute has no visual effect in ordinary web browsers, but can be used by screen readers.

Attributes

ValueDescription
text A short description of the header cell content.

Examples

Name Description Taste
Apple Fruit Sweet and sour
Banana Fruit Sweet
<table>
<tr>

<th abbr="Name">Name</th>
<th abbr="Desc">Description</th>
<th abbr="Taste">Taste</th>

</tr>
<tr>

<td>Apple</td>
<td>Fruit</td>
<td>Sweet and sour</td>

</tr>
<tr>

<td>Banana</td>
<td>Fruit</td>
<td>Sweet</td>

</tr>
</table>

align

Descriptions:
The align attribute specifies the horizontal alignment of the content in a header cell.
Notes:
The align attribute of <th> is not supported in HTML5. Use CSS instead.

Attributes

ValueDescription
left Left-aligns content.
right Right-aligns content.
center Default. Center-aligns content.
justify Stretches the lines so that each line has equal width (like in newspapers and magazines).
char Aligns the content to a specific character.

Examples

Name Description Taste
Apple Fruit Sweet and sour
Banana Fruit Sweet
<table>
<tr>

<th>Name</th>
<th>Description</th>
<th align="left">Taste</th>

</tr>
<tr>

<td>Apple</td>
<td>Fruit</td>
<td>Sweet and sour</td>

</tr>
<tr>

<td>Banana</td>
<td>Fruit</td>
<td>Sweet</td>

</tr>
</table>

bgcolor

Descriptions:
The bgcolor attribute specifies the background color of a header cell.
Notes:
The bgcolor attribute of <th> is not supported in HTML5. Use CSS instead.

Attributes

ValueDescription
color_name Specifies the background color with a color name (like "red").
hex_number Specifies the background color with a hex code (like "#ff0000").
rgb_number Specifies the background color with an rgb code (like "rgb(255,0,0)").

Examples

Apple Fruit Sweet and sour
Banana Fruit Sweet
<table>
<tr>

<th bgcolor="">Name</th>
<th>Description</th>
<th>Taste</th>

</tr>

colspan

Descriptions:
The colspan attribute defines the number of columns a header cell should span.
Notes:
Only Firefox supports colspan="0", which has a special meaning (look below in the "Attribute Values" table).

Attributes

ValueDescription
number Sets the number of columns a header cell should span. Note: colspan="0" tells the browser to span the cell to the last column of the column group (colgroup).

Examples

Fruit list
Name Description Taste
Apple Fruit Sweet and sour
Banana Fruit Sweet
<table>
<tr>

<th colspan="3">Fruit list</th>

</tr>
<tr>

<th>Name</th>
<th>Description</th>
<th>Taste</th>

</tr>
<tr>

<td>Apple</td>
<td>Fruit</td>
<td>Sweet and sour</td>

</tr>
<tr>

<td>Banana</td>
<td>Fruit</td>
<td>Sweet</td>

</tr>
</table>

headers

Descriptions:
The headers attribute specifies one or more header cells a header cell is related to.
The headers attribute has no visual effect in ordinary web browsers, but can be used by screen readers.

Attributes

ValueDescription
header_id Specifies a space-separated list of id's to one or more header cells the header cell is related to.

Examples

Fruit list
Name Description Taste
Apple Fruit Sweet and sour
Banana Fruit Sweet
<table>
<tr>

<th>Name</th>
<th>Description</th>
<th id="taste">Taste</th>

</tr>
<tr>

<td>Apple</td>
<td>Fruit</td>
<td headers="taste">Sweet and sour</td>

</tr>
<tr>

<td>Banana</td>
<td>Fruit</td>
<td headers="taste">Sweet</td>

</tr>
</table>

rowspan

Descriptions:
The rowspan attribute defines the number of rows a header cell should span.
Notes:
Only Firefox and Opera 12 (and earlier versions) support rowspan="0", which has a special meaning (look below in the "Attribute Values" table).

Attributes

ValueDescription
number Sets the number of rows a header cell should span. Note: rowspan="0" tells the browser to span the cell to the last row of the table section (thead, tbody, or tfoot)

scope

Descriptions:
The scope attribute specifies whether a header cell is a header for a column, row, or group of columns or rows.
The scope attribute has no visual effect in ordinary web browsers, but can be used by screen readers.

Attributes

ValueDescription
col Specifies that the cell is a header for a column.
row Specifies that the cell is a header for a row.
colspan Specifies that the cell is a header for a group of columns.
rowspan Specifies that the cell is a header for a group of rows.

Examples

Name Description Taste
Apple Fruit Sweet and sour
Banana Fruit Sweet
<table>
<tr>

<th scope="col">Name</th>
<th scope="col">Description</th>
<th scope="col">Taste</th>

</tr>
<tr>

<td>Apple</td>
<td>Fruit</td>
<td>Sweet and sour</td>

</tr>
<tr>

<td>Banana</td>
<td>Fruit</td>
<td>Sweet</td>

</tr>
</table>
