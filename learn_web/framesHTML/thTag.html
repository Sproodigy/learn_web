Tables

<th>

Descriptions:
The tag defines a header cell in an HTML table.
The text in elements are bold and centered by default.
Notes:
Use the colspan and rowspan attribute to let the content span over multiple columns or rows!

abbr

Descriptions:
The abbr attribute specifies a shorter version of the content in a header cell.
The abbr attribute has no visual effect in ordinary web browsers, but can be used by screen readers.

Attributes

ValueDescription
text A short description of the header cell content.

Examples

Name Description Taste
Apple Fruit Sweet and sour
Banana Fruit Sweet
<table>
<tr>

<th abbr="Name">Name<th>
<th abbr="Desc">Description<th>
<th abbr="Taste">Taste<th>

</tr>
<tr>

<td>Apple<td>
<td>Fruit<td>
<td>Sweet and sour<td>

</tr>
<tr>

<td>Banana<td>
<td>Fruit<td>
<td>Sweet<td>

</tr>
</table>

align

Descriptions:
The align attribute specifies the horizontal alignment of the content in a header cell.
Notes:
The align attribute of <th> is not supported in HTML5. Use CSS instead.

Attributes

ValueDescription
left Left-aligns content.
right Right-aligns content.
center Default. Center-aligns content.
justify Stretches the lines so that each line has equal width (like in newspapers and magazines).
char Aligns the content to a specific character.

Examples

Name Description Taste
Apple Fruit Sweet and sour
Banana Fruit Sweet
<table>
<tr>

<th>Name<th>
<th>Description<th>
<th align="left">Taste<th>

</tr>

rules

Descriptions:
The rules attribute specifies which parts of the inside borders that should be visible.
If the width attribute is not set, a table takes up the space it needs to display the table data.
Notes:
Chrome and Safari displays this attribute incorrectly: They add the affected outside borders in addition to the inside borders.
The width attribute of <table> is not supported in HTML5. Use CSS instead.

Attributes

ValueDescription
none No lines.
groups Lines between row groups and column groups.
rows Lines between rows.
cols Lines between columns.
all Lines between rows and columns.

Examples

Name Description Taste
Apple Fruit Sweet and sour
Banana Fruit Sweet
<table rules="">
<tr>

<th>Name<th>
<th>Description<th>
<th>Taste<th>

</tr>
<tr>

<td>Apple<td>
<td>Fruit<td>
<td>Sweet and sour<td>

</tr>
<tr>

<td>Sweet<td>

</tr>
</table>
