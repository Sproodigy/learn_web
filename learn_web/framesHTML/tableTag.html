Tables

<table>

Descriptions:
The <table> tag defines an HTML table.
An HTML table consists of the <table> element and one or more <tr>, <th>, and >td> elements.
The <tr> element defines a table row, the <th> element defines a table header, and the >td> element defines a table cell.
Notes:
Tables should not be used for page layout! Historically, some Web authors have misused tables in HTML as a way to control their page layout. However, there are a variety of alternatives to using HTML tables for layout, primarily using CSS.

align

Descriptions:
The align attribute specifies the alignment of a table according to surrounding text.
Normally, an HTML table will have a break before and after it. The align attribute allows other HTML elements to wrap around the table.
Notes:
The align attribute of <table> is not supported in HTML5. Use CSS instead.

Attributes

ValueDescription
left Left-aligns the table.
center Center-aligns the table.
right Right-aligns the table.

Examples

Name Description Taste
Apple Fruit Sweet and sour
Banana Fruit Sweet
<table align="left">
<tr>

<th>Name<th>
<th>Description<th>
<th>Taste<th>

</tr>
<tr>

<td>Apple<td>
<td>Fruit<td>
<td>Sweet and sour<td>

</tr>
<tr>

<td>Banana<td>
<td>Fruit<td>
<td>Sweet<td>

</tr>
</table>

rules

Descriptions:
The rules attribute specifies which parts of the inside borders that should be visible.
If the width attribute is not set, a table takes up the space it needs to display the table data.
Notes:
Chrome and Safari displays this attribute incorrectly: They add the affected outside borders in addition to the inside borders.
The width attribute of <table> is not supported in HTML5. Use CSS instead.

Attributes

ValueDescription
none No lines.
groups Lines between row groups and column groups.
rows Lines between rows.
cols Lines between columns.
all Lines between rows and columns.
