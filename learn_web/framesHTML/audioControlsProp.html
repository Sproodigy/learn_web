<controls>

Description:
The controls attribute is a boolean attribute.
When present, it specifies that audio controls should be displayed.
Audio controls should include: play, pause, seeking, volume.
Note:
In XHTML, attribute minimization is forbidden, and the controls attribute must be defined as <audio controls="controls">.

Examples

<audio controls>

<source src="..sounds/cat_mewing.mp3" type="audio/mpeg">
Your browser does not support the audio element.

</audio>
