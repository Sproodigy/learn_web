Styles and Semantics

<span>

Descriptions:
The <span> tag is used to group inline-elements in a document.
The <span> tag provides no visual change by itself.
The <span> tag provides a way to add a hook to a part of a text or a part of a document.
Notes:
When a text is hooked in a <span> element, you can style it with CSS, or manipulate it with JavaScript.

Examples

This is a red word and this is a blue word.

<p> This is a <span style="color: red;"> red /span> word and this is a <span style="color: blue;"> blue </span> word.</p>
