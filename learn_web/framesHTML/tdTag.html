Tables

<td>

Descriptions:
The <td> tag defines a standard cell in an HTML table.
The text in <td> elements are regular and left-aligned by default.
Notes:
Use the colspan and rowspan attribute to let the content span over multiple columns or rows!

colspan

Descriptions:
The colspan attribute defines the number of columns a cell should span.
Notes:
Only Firefox supports colspan="0", which has a special meaning (look below in the "Attribute Values" table).

Attributes

ValueDescription
number Specifies the number of columns a cell should span. Note: colspan="0" tells the browser to span the cell to the last column of the column group (colgroup).

Examples

Name Description Taste
Apple Fruit Sweet and sour
Banana Fruit Sweet
Good fruits!

<tr>

<th>Fruit list</th>

</tr>
<tr>

<th>Name</th>
<th>Description</th>
<th>Taste</th>

</tr>
<tr>

<td>Apple</td>
<td>Fruit</td>
<td>Sweet and sour</td>

</tr>
<tr>

<td>Banana</td>
<td>Fruit</td>
<td>Sweet</td>

</tr>
<tr>

<td colspan="3">Banana</td>

</tr>
</table>

headers

Descriptions:
The headers attribute specifies one or more header cells a header cell is related to.
The headers attribute has no visual effect in ordinary web browsers, but can be used by screen readers.

Attributes

ValueDescription
header_id Specifies a space-separated list of id's to one or more header cells the header cell is related to.

Examples

Fruit list
Name Description Taste
Apple Fruit Sweet and sour
Banana Fruit Sweet
<table>
<tr>

<th>Name</th>
<th>Description</th>
<th id="taste">Taste</th>

</tr>
<tr>

<td>Apple</td>
<td>Fruit</td>
<td headers="taste">Sweet and sour</td>

</tr>
<tr>

<td>Banana</td>
<td>Fruit</td>
<td headers="taste">Sweet</td>

</tr>
</table>

rowspan

Descriptions:
The rowspan attribute defines the number of rows a header cell should span.
Notes:
Only Firefox and Opera 12 (and earlier versions) support rowspan="0", which has a special meaning (look below in the "Attribute Values" table).

Attributes

ValueDescription
number Sets the number of rows a header cell should span. Note: rowspan="0" tells the browser to span the cell to the last row of the table section (thead, tbody, or tfoot)

Examples

Name Description Taste
Apple Fruit Sweet
Banana Fruit
<table>
<tr>

<th>Fruit list</th>

</tr>
<tr>

<th>Name</th>
<th>Description</th>
<th>Taste</th>
<th>Great fruits</th>

</tr>
<tr>

<td>Apple</td>
<td>Fruit</td>
<td rowspan="2">Sweet and sour</td>

</tr>
<tr>

<td>Banana</td>
<td>Fruit</td>

</tr>
</table>
