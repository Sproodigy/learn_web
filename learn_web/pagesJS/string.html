The toLocaleUpperCase() method does not change the original string.

Examples

We animals are the MOST complicated things in the KNOWN UNIVERSE.


<body>
<div>

<p id="text"> We animals are the most... </p>
<button type="button" onclick="setToLocaleUpperCase()"> Perform </button>
<p id="target"> </p>

</div>
</body>
<script>
function setToLocaleUpperCase() {

var string = document.getElementById("text").innerHTML;
var target = document.getElementById("target");
var result = string.toLocaleUpperCase();
target.innerHTML = "Result: " + result;

}
</script>

Description:
The toString() method returns the value of a String object.

Examples

Data: 1234567890
Type: number


<body>
<div>

<p> Data: 1234567890<br>Type: number </p>
<button type="button" onclick="changeToString()"> Perform </button>
<p id="target"> </p>

</div>
</body>
<script>
function changeToString() {

var number = 1234567890;
var target = document.getElementById("target");
var result = number.toString();
var typeOfResult = typeof(result);
target.innerHTML = "Result: " + result + "<br>Type: " + typeOfResult;

}
</script>
