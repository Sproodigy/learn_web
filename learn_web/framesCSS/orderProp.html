Description:
The order property specifies the order of a flexible item relative to the rest of the flexible items inside the same container.
Notes:
If the element is not a flexible item, the order property has no effect.
Internet Explorer 10 and earlier versions do not support the justify-content property.
The following browsers need a prefix: Chrome 21.0 and Safari 6.1 (-webkit-), Firefox 18.0 (-moz-).
Attribute values
ValueDescription
number Default value 0. Specifies the order for the flexible item.
initial Sets this property to its default value.
inherit Inherits this property from its parent element.

Examples

<div>

<span id="orderRed" style="background-color: red;"></span>
<span id="orderYellow" style="background-color: yellow;"></span>
<span id="orderBlue" style="background-color: blue;"></span>
<span id="orderBrown" style="background-color: brown;"></span>

</div>
#orderRed { order: 1; }
#orderYellow { order: 2; }
#orderBlue { order: 3; }
#orderBrown { order: 4; }
span {

width: 70px;
height: 70px;

}
