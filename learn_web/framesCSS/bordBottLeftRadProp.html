Description:
The border-bottom-left-radius property defines the radius of the bottom-left corner.
Note:
This property allows you to add rounded borders to elements!
If you set two values, the first one is for the bottom border, and the second one for the left border. If the second value is omitted, it is copied from the first. If either length is zero, the corner is square, not rounded.
The following browsers need a prefix: Safari 3.4, Chrome 4.0 (-webkit-), Mozilla 3.0 (-moz-).
Attribute values
ValueDescription
length Defines the shape of the bottom-left corner. Default value is 0.
% Defines the shape of the bottom-left corner in %.
initial Sets this property to its default value.
inherit Inherits this property from its parent element.

Examples

<div></div>

div {

border-bottom-left-radius: ?;
border-bottom-width: 3px;

}
