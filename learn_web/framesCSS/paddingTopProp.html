Description:
The padding-to property sets the top padding (space) of an element.
Note:
Negative values are allowed.
Attribute values
ValueDescription
length Specifies a fixed top padding in px, cm, em, etc. Default value is 0. Negative values are allowed.
% Specifies a top padding in percent of the width of the element.
initial Sets this property to its default value.
inherit Inherits this property from its parent element.
