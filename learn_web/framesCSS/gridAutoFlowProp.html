Description:
The grid-auto-flow property controls how auto-placed items get inserted in the grid.
Note:
Add the value "dense" to fill any holes in the grid.
Attribute values
ValueDescription
row Default value. Insert auto-placed items by filling each row.
column Insert auto-placed items by filling each column.
row dense Insert auto-placed items by filling each row, and fill any holes in the grid.
column dense Insert auto-placed items by filling each column, and fill any holes in the grid.
