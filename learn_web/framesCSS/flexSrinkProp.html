Description:
The flex-shrink property specifies how the item will shrink relative to the rest of the flexible items inside the same container.
Note:
If the element is not a flexible item, the flex-shrink property has no effect.
The following browsers need a prefix: Chrome 21.0 and Safari 6.1 (-webkit-), Firefox 18.0 (-moz-).
Internet Explorer 10 and earlier versions do not support the flex-basis property.
Attribute values
ValueDescription
number A number specifying how much the item will shrink relative to the rest of the flexible items. Default value is 1.
initial Sets this property to its default value.
inherit Inherits this property from its parent element.

Examples

<div>
<p background-color: blue;> </p>
<p background-color: green;> </p>
<p id="example" > background-color: yellow;> </p>
<p background-color: lightblue;> </p>
<p background-color: coral;> </p>
</div>

div {

display: flex;

}

div p {
flex: 1 1 200px;
}

#example {
flex-shrink: 1;
}
