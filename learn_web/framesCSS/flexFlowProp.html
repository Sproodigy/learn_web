Description:
The flex-flow property is a shorthand property for: flex-direction, flex-wrap.
Note:
If the element is not a flexible item, the flex-flow property has no effect.
The following browsers need a prefix: Chrome 21.0 and Safari 6.1 (-webkit-), Firefox 18.0 (-moz-).
Internet Explorer 10 and earlier versions do not support the flex-flow property.
Attribute values
ValueDescription
flex-direction Specifying the direction of the flexible items.
flex-wrap Specifying whether the flexible items should wrap or not.
initial Sets this property to its default value.
inherit Inherits this property from its parent element.
