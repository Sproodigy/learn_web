Description:
The transform-style property specifies how nested elements are rendered in 3D space.
Note:
This property must be used together with the transform property.
Numbers followed by -webkit-, -moz-, or -o- specify the first version that worked with a prefix:
Chrome 12.0, Safari 4.0 and Opera 15.0 (-webkit-), Firefox 10.0(-moz-).
Attribute values
ValueDescription
flat Specifies that child elements will NOT preserve its 3D position. This is default.
preserve-3d Specifies that child elements will preserve its 3D position.
initial Sets this property to its default value.
inherit Inherits this property from its parent element.

Examples

Original
<div>
<div id="div2">
 <div id="div3"></div>
</div>
</div>

div2 {

width: 300px;
margin: 30px auto;
padding: 10px;
border: 2px solid blue;
background-color: red;
transform: rotateX(30deg);
transform: rotateY(60deg);
transform-style: ?;

}

div3 {

padding: 20px;
border: 2px solid blue;
background-color: yellow;
transform: rotateY(-40deg);

}
