Description:
The flex-basis property specifies the initial length of a flexible item.
Note:
If the element is not a flexible item, the flex-basis property has no effect.
The following browsers need a prefix: Chrome 21.0 and Safari 6.1 (-webkit-), Firefox 18.0 (-moz-).
Internet Explorer 10 and earlier versions do not support the flex-basis property.
Attribute values
ValueDescription
number A number specifying how much the item will grow relative to the rest of the flexible items.
auto A number specifying how much the item will shrink relative to the rest of the flexible items.
initial Sets this property to its default value.
inherit Inherits this property from its parent element.
