Description:
The object-fit property is used to specify how an <img> or <video> should be resized to fit its container.
This property tells the content to fill the container in a variety of ways; such as "preserve that aspect ratio" or "stretch up and take up as much space as possible".
The object-fit property is not supported in Internet Explorer/Edge 15 or earlier.
Attribute values
ValueDescription
none The replaced content is not resized.
fill This is default. The replaced content is sized to fill the element's content box. If necessary, the object will be stretched or squished to fit.
contain The replaced content is scaled to maintain its aspect ratio while fitting within the element's content box.
cover The replaced content is sized to maintain its aspect ratio while filling the element's entire content box. The object will be clipped to fit.
scale-down The content is sized as if none or contain were specified (would result in a smaller concrete object size).
initial Sets this property to its default value.
inherit Inherits this property from its parent element.

Examples

<div>
<img src="../images/mountain.jpg">
 <img id="fitImg" src="../images/mountain.jpg">
 </div>

div {

max-width: 85%;
border: 1px solid white;
margin-left: 2em;
padding: 30px 0;
overflow: auto;
font-size: 20px;
text-align: center;

}

img {

width: 450px;
height: 300px;

}

#fitImg {

width: 150px;
height: 300px;
object-fit: ?;

}
