Description:
The transition property is a shorthand property for: transition-property, transition-duration, transition-timing-function, transition-delay.
Note:
This transition-duration property does not work in Internet Explorer 9 and earlier versions.
Numbers followed by -webkit-, -moz-, or -o- specify the first version that worked with a prefix:
two value â€” Chrome 4.0 and Safari 3.1, Firefox 4.0 (-moz-), Opera 10.5 (-o-).
Attribute values
ValueDescription
transition-property Specifies the name of the CSS property the transition effect is for.
transition-duration Specifies how many seconds or milliseconds the transition effect takes to complete.
transition-timing-function Specifies the speed curve of the transition effect.
transition-delay Defines when the transition effect will start.
initial Sets this property to its default value.
inherit Inherits this property from its parent element.

Examples

#div1 {

width: 100px;
height: 100px;
padding: 10px;
background: blue;
transition: ?;

}

#div1:hover {

width: 200px;
height: 200px;
opacity: 0.3;

}
