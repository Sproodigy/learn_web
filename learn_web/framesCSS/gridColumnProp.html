Description:
The grid-column property specifies a grid item's size and location in a grid layout, and is a shorthand property for the following properties: grid-column-start, grid-column-end.
Attribute values
ValueDescription
grid-column-start Specifies on which column to start displaying the item.
grid-column-end Specifies on which column-line to stop displaying the item, or how many columns to span.

Examples

1 2 3 4 5 6

<span class="item6">6</span>

</div>
