Description:
The grid-gap property defines the size of the gap between the rows and columns in a grid layout, and is a shorthand property for the following properties: grid-row-gap, grid-column-gap.
Attribute values
ValueDescription
grid-row-gap Sets the size of the gap between the rows in a grid layout. 0 is the default value.
grid-column-gap Sets the size of the gap between the columns in a grid layout. 0 is the default value.
