Description:
The font-kerning property controls the usage of the kerning information stored in a font.
Kerning defines how letters are spaced.
Note:
For fonts that do not include kerning data, this property will have no visible effect.
The following browsers need a prefix: Chrome 32.0 (-webkit-), Opera ?.
Internet Explorer 9 and earlier versions do not support the font-kerning property.
Attribute values
ValueDescription
auto Default. The browser determines whether font kerning should be applied or not.
normal Specifies that font kerning is applied.
none Specifies that font kerning is not applied.
