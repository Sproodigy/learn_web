Description:
The padding property sets the paddings for an element, and is a shorthand property for the following properties: padding-top, padding-right, padding-bottom, padding-left.
If the padding property has four values: top, right, bottom, left.
If the padding property has three values: top, right and left, bottom.
If the padding property has two values: top and bottom, right and left.
If the padding property has one value: all four margins.
Note:
Negative values are allowed.
Top and bottom margins of elements are sometimes collapsed into a single margin that is equal to the largest of the two margins.
This does not happen on horizontal (left and right) margins! Only vertical (top and bottom) margins!
Attribute values
ValueDescription
length Specifies a margin in px, cm, em, etc. Default value is 0. Negative values are allowed.
% Specifies a margin in percent of the width of the containing element.
initial Sets this property to its default value.
inherit Inherits this property from its parent element.
