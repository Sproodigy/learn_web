Description:
The page-break-inside property sets whether a page-break should be avoided inside a specified element.
Note:
You cannot use this property on an empty <div> or on absolutely positioned elements.
Attribute values
ValueDescription
auto Default. Automatic page-break.
avoid Avoid a page-break inside the element (if possible).
initial Sets this property to its default value.
inherit Inherits this property from its parent element.

Examples

@media print {

pre, blockquote { page-break-inside: avoid; }

}
