Description:
The isolation property defines whether an element must create a new stacking content.
Note:
The isolation property is helpful when used with background-blend-mode or mix-blend-mode.
Edge/Internet Explorer do not support the isolation property.
Attribute values
ValueDescription
auto Default. A new stacking context is created only if one of the properties applied to the element requires it.
isolate A new stacking context must be created.
initial Sets this property to its default value.
inherit Inherits this property from its parent element.

Examples

isolationDivD: isolation: auto;
isolationDivE: isolation: isolate;
<div class="isolationDivA">

<div id="isolationDivD">
<div class="isolationDivA isolationDivC">
isolationDivD: isolation: auto;
</div>
</div>
<div id="isolationDivE">
<div class="isolationDivA isolationDivC">
isolationDivD: isolation: isolate;
</div>
</div>

</div>

.isolationDivA {

background-color: blue;

}

.isolationDivC {

width: 150px;
height: 150px;
border: 1px solid white;
padding: 2px;
mix-blend-mode: difference;

}

#isolationDivD {

isolation: auto;

}

#isolationDivE {

isolation: isolation;

}
