Description:
An outline is a line that is drawn around elements, outside the borders, to make the element "stand out".
The outline property is a shorthand property for: outline-width, outline-style (required), outline-color.
If outline-color is omitted, the color applied will be the color of the text.
Note:
The outline is not a part of the element's dimensions, therefore the element's width and height properties do not contain the width of the outline.
Attribute values
ValueDescription
outline-width Specifies the width of outline.
outline-style Specifies the style of the outline.
outline-color Specifies the color of the outline.
initial Sets this property to its default value.
inherit Inherits this property from its parent element.
