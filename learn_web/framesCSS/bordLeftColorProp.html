Description:
The border-left-color property sets the color of an element's left border.
Note:
Always declare the border-style or the border-left-style property before the border-left-color property. An element must have a border before you can change the color.
Attribute values
ValueDescription
color Specifies the color of the left border. Default color is the color of the element.
transparent Specifies that the border color should be transparent.
initial Sets this property to its default value.
inherit Inherits this property from its parent element.

Examples

<div></div>

div {

border-left-width: 3px;
border-left-color: ?;

}
