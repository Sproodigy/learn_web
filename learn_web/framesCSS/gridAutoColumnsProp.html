Description:
The grid-auto-columns property sets a size for the columns in a grid container.
Note:
This property affects only columns with the size not set.
The grid-template-columns property overrides this property.
Attribute values
ValueDescription
auto Default value. The size of the columns is determined by the size of the container.
max-content Sets the size of each column to depend on the largest item in the column.
min-content Sets the size of each column to depend on the largest item in the column.
length Sets the size of the columns, by using a legal length value.

Examples

1 2 3 4 5 6
<div>

<span class="item1">1</span>
<span class="item2">2</span>
<span class="item3">3</span>
<span class="item4">4</span>
<span class="item5">5</span>
<span class="item6">6</span>

</div>
