elapsedTime

Description:
The elapsedTime property returns the number of seconds an animation has been running, when an animation event occurs.
This property is read-only.
Notes:
The return value is not affected if the animation is paused (by using the animation-delay CSS property).
For the animationstart event, this property always returns "0".

Examples

<body>
<div id="eventElapsedTime"> </div>
</body>
<style>
#eventElapsedTime {

width: 350px;
height: 100px;
background-color: green;
position: relative;
-webkit-animation-name: someAnimation;
-webkit-animation-duration: 3s;
animation-name: someAnimation;
animation-duration: 3s;

}
</style>
<script>

var x = document.getElementById("eventElapsedTime");
x.addEventListener("webkitAnimationIteration", elapsedTimeFunction);
x.addEventListener("animationiteration", elapsedTimeFunction);

function elapsedTimeFunction(event) {

this.innerHTML = "Elapsed time: " + event.elapsedTime;

}
</script>
