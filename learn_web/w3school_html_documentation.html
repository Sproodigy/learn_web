1/ Required tags and rules

Tip: HTML tags normally come in pairs. The start tag is also called the opening tag, and the end tag the closing tag.
Note: Metadata typically define the document title, character set, styles, links, scripts, and other meta information.
Note: Do not forget the end tags. It might produce unexpected results and/or errors if you forget the end tag.
Note: Use lowercase tags and always 'closed' the empty tags, like <br /> (defines a line break).
Note: Only the content inside the <body> section is displayed in a browser.

Page view

<!DOCTYPE html>
<html>

<head>
<title> Page title </title>
< meta charset ="utf-8" / >
</head>

<body>
<h1> Heading </h1>
<p> Paragraph </p>
</body>

</html>

Explane

The <!DOCTYPE html> declaration defines this document to be HTML5
The <html> element is the root element of an HTML page

The <head> element contains meta information about the document
The <title> element specifies a title for the document



The <body> element contains the visible page content
The <h1> element defines a large heading
The <p> element defines a paragraph

2/ Attributes

Description

  • Attributes provide additional information about an element
  • Attributes are always specified in the start tag
  • Attributes usually come in name/value pairs like: name="value"

5/ Styles

Note: The style nested directly in the tag overwrites all other external styles.

Syntax

< tagname style = "property: value;">

Examples

<p style = "color: slateblue;"> Color is slateblue <p> ⇩ Color is slateblue <p style = "font-family: courier;"> Font family is courier <p> ⇩ Font family is courier

6/ Text formatting

Note: Browsers display <strong> as <b>, and <em> as <i>. However, there is a difference in the meaning of these tags: <b> and <i> defines bold and italic text, but <strong> and <em> means that the text is "important".

Examples

<b>Bold text</b>
<strong>Important text</strong>
<i>Italic text</i>
<em>Emphasized text<em> <mark>Default<mark>
<mark style ="background-color: violet;"> Marked text</mark>
<small>Small text<small> <del>Deleted text</del>
<ins>Inserted text</ins>
<sub>Subscript text</sub>
<sup>Superscript text</sup>

10/ Links

Note: A link does not have to be text. It can be an image or any other HTML element.
Note: Without a forward slash at the end of subfolder addresses, you might generate two requests to the server. Many servers will automatically add a forward slash to the end of the address, and then create a new request.
Tip: If your webpage is locked in a frame, you can use target="_top" to break out of the frame.
Note: If no frames, any value works like _self.

Examples

The href attribute

<a href= "IMG-7719c7d66277e5df8ae206377550abc6-V.jpg"> This is a link in the folder <a>
<a href= "http://fb.ru/misc/i/gallery/58679/2351462.jpg"> This is a link on the Interner <a>

The target attribute

<a href= "IMG-7719c7d66277e5df8ae206377550abc6-V.jpg" target="_blank"> Opens the linked document in a new window or tab <a>
<a href= "IMG-7719c7d66277e5df8ae206377550abc6-V.jpg" target="_self"> Opens the linked document in the same window/tab as it was clicked (this is default) <a>
<a href= "IMG-7719c7d66277e5df8ae206377550abc6-V.jpg" target="_parent"> Opens the linked document in the parent frame <a>
<a href= "IMG-7719c7d66277e5df8ae206377550abc6-V.jpg" target="_top"> Opens the linked document in the full body of the window <a>
<a href= "IMG-7719c7d66277e5df8ae206377550abc6-V.jpg" target="framename"> Opens the linked document in a named frame <a>

11/ Images

Note: The alt attribute is required. A web page will not validate correctly without it.
Note: Always specify the width and height of an image. If width and height are not specified, the page might flicker while the image loads. Using the 'style' attribute prevents styles sheets from changing the size of images.

12/ Tables

Note: The elements are the data containers of the table. They can contain all sorts of HTML elements; text, images, lists, other tables, etc.
Note: If the table has collapsed borders, border-spacing has no effect.
Note: The <caption> tag must be inserted immediately after the <table> tag.

Example

May

</table>

14/ Block and inline elements

Note: A block-level element always starts on a new line and takes up the full width available (stretches out to the left and right as far as it can).
Note: An inline element does not start on a new line and only takes up as much width as necessary.

Examples

The <div> and <p> elements

<div style= "background-color: lime; font-family: verdana;">

<p> London is the capital city of England. It is the most populous city in the United Kingdom, with a metropolitan area of over 13 million inhabitants. </p>

</div>

The <span> element

<div style= "background-color: lime; font-family: verdana;">

<p> London is the capital city of England. It is the
< span style ="background-color: yellow; color: red;"> most populous city in the United Kingdom, with a metropolitan area of over 13 million inhabitants. </span> </p>

</div>

Note: The class name is case sensitive!

16/ Id

Tip: The id attribute can be used on any HTML element.
Note: The id name is case sensitive!
An HTML element can only have one unique id that belongs to that single element, while a class name can be used by multiple elements.

Examples

Example 1

CSS (internal)

<style>
#myHeader {
background-color: tomato;
color: black;
padding: 40px;
text-align: center;
}
</style>

HTML

<h1 id="myHeader">My Header<h1>
 ⇩

My Header

Bookmarks

<h1 id= "C1"<h1>
<a href= "#C1">Jump to Chapter 1<a>
